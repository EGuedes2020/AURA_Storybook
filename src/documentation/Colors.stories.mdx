# Colors

Esta é a paleta de cores da aplicação. As palavras de baixo de cada cor (ex: *"mint_vivid"*, *"danger"*, *"blue_dark"*), são as **strings** que deverão ser utilizadas para testar as cores.   

Elas estão normalmente presentes como **"presetColors"** no *color picker* para facilitar os testes. Caso haja uma anormalidade, utilize as **strings** referidas.
